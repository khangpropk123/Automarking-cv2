# Automarking-cv2

Auto marking multiple choice using openCV.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install lib.

```bash
pip install opencv-python
pip install numpy
pip install matplotlib
```

## Usage
Must be change src of image
```python
python grade.py
```
